📊 TimeTrackingAPI — Учет рабочего времени
Система для учета рабочего времени сотрудников по проектам и задачам.

🎯 Для чего нужен проект
Позволяет командам и компаниям:

Отслеживать время, потраченное на проекты и задачи

Контролировать загрузку сотрудников (не более 24 часов в день)

Анализировать эффективность работы по дням/месяцам

Визуализировать статус дня (недостаточно/достаточно/избыточно)

⚙️ Основные возможности
Проекты — создание, редактирование, удаление, статус "Активный"

Задачи — привязка к проектам, статус "Активный". Автоматически деактивируются вместе с проектом

Проводки — учет времени по задачам. Фильтрация: все записи / за день / за месяц. Ограничение — не более 24 часов в день суммарно по всем задачам

Визуализация — цветовой стикер за день: 🟡 желтый, если отработано менее 8 часов; 🟢 зеленый, если ровно 8 часов; 🔴 красный, если более 8 часов
